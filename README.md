# 프로젝트 소개

## 🎯 ADS
나의 일정으로 반려동물을 돌보지 못 할 때, 글을 올려서 펫을 잠시 맡아줄 메이트를 찾을 수 있습니다. 

## ✅ 주요기능
- 나의 펫을 일정 기간동안 맡기는(산책/돌봄) 글을 올립니다.
- 메이트 신청자와 채팅을 통해 약속을 잡고, 리뷰를 남깁니다. 
- 나의 펫과 함께 갈 수 있는 장소들을 서로 공유하고 지도에서 확인합니다. 

## 📝 작동환경
- Xcode 16.0
- iOS 18.0
